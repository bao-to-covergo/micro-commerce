# Integrations

How to configure authentication and authorization for the Flux integrations with external systems.

As the leading project in cloud-native GitOps, Flux integrates with several external systems to provide a complete GitOps experience, like Git repositories, OCI artifact registries, object storage, key management services and notification systems. This section describes how Flux can be configured to communicate with and get access to resources on these external systems.
