 _string_|  Name of the referent If multiple resources are targeted `*` may be set.  
`namespace`  
 _string_|  _(Optional)_ Namespace of the referent  
`matchLabels`  
 _map[string]string_|  _(Optional)_ MatchLabels is a map of {key,value} pairs. A single {key,value} in the matchLabels map is equivalent to an element of matchExpressions, whose key field is "key", the operator is "In", and the values array contains only "value". The requirements are ANDed. MatchLabels requires the name to be set to `*`.
