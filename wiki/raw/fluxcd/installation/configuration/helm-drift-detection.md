# Flux drift detection for Helm Releases

How to enable Helm drift detection in Flux

Since Flux v2.2.0, drift detection for Helm releases is available as a configuration option on the HelmRelease itself. Refer to the [HelmRelease documentation](</flux/components/helm/helmreleases/#drift-detection>) for more information.
